# Misc Folder Index

This folder stores project notes and tracking files that support the CDA rewrite but are not part of the paper source.

## Subfolders

| Folder | Purpose |
|---|---|
| `method/` | Final method decisions and implementation-name mapping. |
| `results/` | Current result trackers, accepted result summaries, and planned table/figure registry files. |
| `evidence/` | Evidence schemas and templates for raw-result provenance. |
| `audits/` | Audits of old manuscript claims and old local result files. |
| `literature_notes/` | Literature style notes and modern related-work motivation notes. |
| `panel_reviews/` | Reviewer-panel records and critique logs. |

## Current Key Files

| File | Use |
|---|---|
| `method/final_method.md` | Records \(\mathrm{CDA}=\mathrm{VSC}+\mathrm{CDA\text{-}BD}\). |
| `method/paper_skeleton.md` | Planned paper structure, figures, tables, and rationale. |
| `results/cda_results_tracking.md` | Human-readable current results and future result template tables. |
| `results/cda_results_tracking.csv` | Machine-readable current results and fill-in rows. |
| `results/result_registry.yml` | Planned manuscript claims, expected outputs, and validation gates. |
| `evidence/result_evidence_schema.md` | Required fields for publication-grade result evidence. |
| `evidence/result_evidence_template.csv` | Empty evidence-row template. |
| `audits/old_results_audit.md` | Human-readable audit of old unsupported or provisional claims. |
| `audits/old_result_inventory.csv` | Machine-readable inventory of old empirical claims. |
| `literature_notes/modern_literature_refresh.md` | Current motivation and related-work support notes. |
| `literature_notes/literature_style_audit.md` | Writing-style and structure notes from downloaded paper sources. |
