\section*{Reproducibility}
An executable version of our paper, hosted on CodaLab, can be found at \url{https://wilds.stanford.edu/codalab}. This contains the exact commands, code, environment, and data used for the experiments reported in our paper, as well as all trained model weights. The WILDS package is open-source and can be found at \url{https://github.com/p-lambda/wilds}.
